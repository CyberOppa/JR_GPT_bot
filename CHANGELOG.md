## [1.0.0] - 2026-03-18

### Added
- **RAG URL Processing**: The RAG system can now process web links (news, articles, forums) to use as a knowledge source.
- **RAG Audio Answers**: Added a "Read" button to get a text-to-speech audio version of the RAG answer.

### Changed
- **UI/UX**: The "Read" button in RAG mode now only appears when there is an answer to be read.
- **UI/UX**: Removed the redundant "Clear source" button from the RAG keyboard, simplifying the interface. "New source" now handles this functionality.

### Fixed
- The audio response message in RAG mode no longer shows a "Read" button.
